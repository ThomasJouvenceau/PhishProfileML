# Plan Video

## Duree cible

Entre 5 et 15 minutes.

## Plan recommande

### 1. Introduction

- presentation du sujet ;
- lien entre cybersecurite et personnalisation ;
- objectif de PhishProfileML.

### 2. Problematique

- pourquoi une campagne generique peut etre limitee ;
- hypothese : la personnalisation pilotee par ML peut mieux fonctionner.

### 3. Donnees de profil

- panel de 26 etudiants ;
- anonymisation ;
- variables retenues ;
- transformation vers un dataset exploitable.

### 4. Pipeline technique

- preprocessing ;
- encodage ;
- clustering KMeans ;
- attribution d'un segment principal ;
- recommandation de categorie de message.

### 5. Demonstration

- lancement du pipeline ;
- affichage de `processed_profiles.csv` ;
- affichage de `profiles_segmented.csv` ;
- demonstration de l'app Streamlit si elle est utilisee.

### 6. Evaluation

- presentation du protocole experimental ;
- comparaison campagne generique vs personnalisee ;
- metriques retenues.

### 7. Resultats

- graphiques principaux ;
- tableau de comparaison ;
- performance par segment ;
- validation ou non de l'hypothese.

### 8. Limites et conclusion

- limites de l'echantillon ;
- limites du modele ;
- pistes d'amelioration.

## Repartition orale possible
