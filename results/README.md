# Resultats

Ce dossier regroupe les sorties finales du projet.

Fichiers principaux :
- `profiles_segmented.csv` : sortie du pipeline de segmentation et de recommandation.
- `summary_metrics.json` : synthese globale des metriques finales.
- `tables/` : tableaux a inserer dans le rapport ou la video.
- `charts/` : graphiques exportes pour la restitution visuelle.

Conseil de remplissage :
- completez d'abord `data/campaign_results.csv` ;
- calculez ensuite les metriques comparees ;
- renseignez `tables/` puis `summary_metrics.json` ;
- exportez enfin les graphiques dans `charts/`.
